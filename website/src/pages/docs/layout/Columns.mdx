# @substance/layout

## Columns

The `Columns` component is used for horizontal layouts of fixed widths. `Columns`
are wrapped when unable to fit on a row.

### Installation

```
yarn add @substance/layout
```

### Usage

```jsx
<Columns>
  <Columns.Column width={{mobile: 1, tablet: 1 / 3, desktop: 1 / 5}}>
    touch
  </Columns.Column>
  <Columns.Column width={{mobile: 1, tablet: 1 / 3, desktop: 1 / 5}}>
    sight
  </Columns.Column>
  <Columns.Column width={{mobile: 1, tablet: 1 / 3, desktop: 1 / 5}}>
    hearing
  </Columns.Column>
  <Columns.Column width={{mobile: 1, tablet: 1 / 3, desktop: 1 / 5}}>
    smell
  </Columns.Column>
  <Columns.Column width={{mobile: 1, tablet: 1 / 3, desktop: 1 / 5}}>
    taste
  </Columns.Column>
</Columns>
```
